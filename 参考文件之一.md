# 📁 SRPPMS 项目完整目录结构清单

> **项目名称**: SRPPMS (Scientific Research Project Management System)  
> **项目描述**: 基于 SpringBoot + Vue + Sa-Token 的前后端分离科研项目管理平台  
> **技术栈**: Spring Boot 2.7.0 + Vue2 + MyBatis-Plus + Redis + MySQL  
> **生成时间**: 2025-08-19

---
